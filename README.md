- Metamask

- Infuria
- ReactJs
- Tailwind CSS
- Ganache-CLI
- Solidity
- Web3js
- Faucet

## Running the demo

To run the demo follow these steps:

1. Install yarn in cmd or terminal and then run `yarn install`
  <br>
3. Run the app using `yarn start`
